    Graphic File Types

  • Pixels:They are tiny dots of color known as pixels and each digital image is comprised of thousands to millions of individual pixels.

  • Resolution: It is the number of pixels that are displayed per inch of an image and most computer monitors sisplay at the resolutions of 72 pixels per inch.

  • Lossy Compression: It is the result of lost data and quality from the original version and it is associated with files like jpeg and Gif, It does however take up less space on the disk file by lossing some of the data for the image.

  • Lossless Compression: It reduces the file size with no loss of quality and pixels, it can be used for audio files and image files and it rewrites the data of the original file in a more effective way but the file takes uop more space on the disk file then the lossy compression.

  • JPEG: It stands for Joint Photographic Expert Groups, it is a lossy format that uses a 24-bit mage format and uses 16.8 million colors. It is not idea; for images with lots of lines and text due to them lossing data after compression.

  • PNG: Portable Network Graphics, this is a image format that is designed for the web. It takes the best parts of JPEG and GIF and it comes in the two formats of PNG-24 and PNG-8. PNG-8 can display a maximum of 256 colors while PNG-24 format can display millions of colors and is good for photographs.

  • SVG:Scalable Vector Graphics, it is a vector format that is widely popular because of the diffent uses that can come from it. SVG can be modified using CSS and Java Scripts and are very scalable. They are great for reponsive design and is best suited for logos, icons, maps, flags, charts, and other graphics.

  • GIF: Graphics Interchange Format, it can only support 256 colors but it can be used for graphics with a low pixel to pixel variation such as logos and icons.
